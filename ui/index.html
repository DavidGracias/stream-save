Features

Save Movies

Add your favorite movies with custom stream links to your personal library.

Save Series

Store TV series and individual episodes with your preferred streaming sources.

Cloud Storage

Your content is safely stored in MongoDB cloud database, accessible from anywhere.

Available Routes

Stremio API Routes
GET /<user>/<passw>/<cluster>/manifest.json

Returns the addon manifest for Stremio

GET /manifest.json

Same as above, without credentials

GET /<user>/<passw>/<cluster>/stream/<type>/<id>.json

Returns stream data for movies/series

GET /<user>/<passw>/<cluster>/catalog/<type>/<id>.json

Returns catalog metadata

Web Interface Routes
GET/POST /configure

Configuration page for MongoDB setup

GET/POST /manage

Content management interface

GET /

Landing page (redirects to /configure)

GET /<user>/<passw>/<cluster>/configure

Addon configuration (redirects to /manage)
